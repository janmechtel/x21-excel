# Appearance & Customization

X21 provides several customization options to tailor the interface to your workflow.

## Resizable Input Area

### Adjusting Input Height

The input textarea can be resized:

1. Hover over the top edge of the input area
2. Resize handle (dots) appears
3. Click and drag up/down
4. Release to set height

**Benefits**:
- More space for long prompts
- Better view of multi-line requests
- Comfortable typing area

### Auto-Scroll

The input area automatically scrolls as you type multi-line prompts.

All UI components (buttons, panels, text) adapt to the selected theme.

## Panel Layout

### Collapsible Sections

Collapse sections to focus:
- **Thinking blocks**: Click to expand/collapse
- **Tool details**: Click to show/hide parameters
- **History panel**: Toggle with Ctrl+H
- **Tools panel**: Toggle with Ctrl+Shift+T

### Conversation View

Messages display with:
- Clear sender labels (You / Assistant)
- Timestamp for each message
- Expandable content blocks
- Syntax-highlighted code

## Accessibility Features

### Keyboard Navigation

Full keyboard support:
- **Tab**: Move between interactive elements
- **Arrow keys**: Navigate lists
- **Enter**: Activate buttons
- **Escape**: Close overlays

### Tooltips

Hover over buttons and icons for descriptions.

### Focus Management

- Input auto-focuses on startup
- Focus returns to input after actions
- Clear visual focus indicators
