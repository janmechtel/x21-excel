# Conversation History

X21 automatically saves all your conversations and makes them searchable. Access your history to review past work, reuse successful prompts, or continue previous tasks.

## Opening History

Press **Ctrl+H** (or **Cmd+H** on Mac) to toggle the history panel.

The panel slides in from the side showing your recent conversations.

## History Features

### Recent Conversations

Conversations display with:
- **Timestamp**: "Just now", "5m", "2h", "3d", etc.
- **First message**: Preview of the conversation
- **Workbook**: Which file the conversation belongs to

### Search Conversations

Type in the search box to filter by keywords:

```
Search: "sales analysis" → finds all conversations mentioning sales
Search: "pivot table" → finds pivot table discussions
```

### Scope Filtering

Choose conversation scope:
- **File**: Current workbook only
- **All**: Across all workbooks

Switch scope with the dropdown at the top of the history panel.

## Working with History

### Load a Conversation

1. Open history panel
2. Find the conversation
3. Click to load it
4. Previous messages appear
5. Continue where you left off

### New Conversation

Click **New Chat** to start fresh:
- Clears current conversation
- Confirms if ongoing requests exist
- Saves previous conversation automatically

## Search Tips

**By content**:
```
formula
formatting
chart
```

**By date** (use relative terms):
```
today
yesterday
last week
```

**By sheet name**:
```
Sheet1
Sales
Report
```

## Best Practices

### Organize by Task

Start new conversations for different tasks:
- One conversation per analysis project
- New conversation for different data sets
- Fresh start for unrelated work

### Name Your Conversations

While X21 doesn't support manual naming, make your first message descriptive:

```
✓ "Q1 2024 Sales Analysis for Regional Report"
✗ "Analyze this"
```

The first message becomes the conversation preview.

### Use Search Effectively

**Find specific work**:
```
"revenue forecast Q2"
```

**Find by tool used**:
```
"pivot table"
"VBA macro"
```

## Keyboard Shortcuts

- **Ctrl+H** / **Cmd+H**: Toggle history
- **Escape**: Close history panel
- **Arrow keys**: Navigate conversations
- **Enter**: Load selected conversation
