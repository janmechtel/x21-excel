# Keyboard Shortcuts Reference

Quick reference for all X21 keyboard shortcuts. Mac users: Replace `Ctrl` with `Cmd`.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+H` | Toggle conversation history panel |
| `Ctrl+Shift+T` | Toggle tools management panel |
| `Ctrl+N` | Start new conversation |
| `Escape` | Cancel streaming / Close overlays |

## Input Area

| Shortcut | Action |
|----------|--------|
| `Tab` | Send message (when ready) |
| `Shift+Enter` | New line in input |
| `Ctrl+V` | Paste (including files) |
| `@` | Open sheet mention palette |

## Command Palette

| Shortcut | Action |
|----------|--------|
| `/` | Open slash command palette |
| `Arrow Up/Down` | Navigate commands/mentions |
| `Enter` | Select command/mention |
| `Escape` | Close palette |
| Type to search | Filter commands |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Arrow keys` | Navigate lists/palettes |
| `Tab` | Move between UI elements |
| `Enter` | Activate focused element |
| `Escape` | Close panels/overlays |

## Message Actions

| Shortcut | Action |
|----------|--------|
| Click range pill | Navigate to range in Excel |
| Click sheet mention | Switch to that sheet |
| `Ctrl+C` | Copy message content |

## Tool Panels

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+T` | Open/close tools panel |
| Arrow keys | Navigate tool list |
| Space | Toggle tool on/off |
| Search | Filter tools |

## History Panel

| Shortcut | Action |
|----------|--------|
| `Ctrl+H` | Open/close history |
| Arrow keys | Navigate conversations |
| Enter | Load selected conversation |
| Search | Filter by keywords |

## Focus Management

- Input area auto-focuses on startup
- Focus returns to input after tool approvals
- `Tab` cycles through interactive elements
- `Escape` returns focus to input

## Tips

### Efficiency

**Use keyboard for common tasks**:
```
Ctrl+H → Find conversation → Enter → Continue work
```

**Quick tool management**:
```
Ctrl+Shift+T → Search tool → Toggle → Escape
```

### Combinations

Chain shortcuts for workflows:
```
1. Ctrl+N (new chat)
2. @ + sheet name (reference sheet)
3. / + command (execute workflow)
```

## Platform Differences

| Windows/Linux | Mac | Action |
|---------------|-----|--------|
| `Ctrl+H` | `Cmd+H` | Toggle history |
| `Ctrl+Shift+T` | `Cmd+Shift+T` | Toggle tools |
| `Ctrl+N` | `Cmd+N` | New chat |
| `Ctrl+V` | `Cmd+V` | Paste |

## Accessibility

- Full keyboard navigation supported
- No mouse required for any operation
- Screen reader compatible
- Focus indicators visible
