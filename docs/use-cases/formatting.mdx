# Formatting Operations

X21 can read and apply various formatting to your spreadsheets, making data presentation professional and consistent.

## Available Formatting

### Text Formatting

- **Bold**, *italic*, underline
- Font size and font name
- Text color

### Cell Formatting

- Background color
- Border styles
- Alignment (left, center, right, justify)
- Number formats (currency, percentage, dates)

## Common Tasks

### Making Headers Stand Out

```
"Make the first row bold with a light gray background"
```

### Applying Currency Format

```
"Format column B as USD currency with 2 decimal places"
```

### Color Coding

```
"Highlight cells in column C: green if >100, red if <0"
```

### Consistent Styling

```
"/format" - applies professional formatting automatically
```

## Best Practices

1. **Read first**: Let AI analyze current formatting
2. **Be specific**: State colors, sizes, exact formats
3. **Preview changes**: Use View before applying
4. **Batch operations**: Format multiple ranges at once

## Tools Used

- `read_format` - Inspect current formatting
- `write_format` - Apply new formatting
